# Contributing

Small fixes are welcome. This project is maintained as a compact static study page, so contributions should stay focused and easy to review.

Good contributions include:

- Fixing typos or unclear material-processing explanations
- Adding verified review cards
- Improving mobile layout without adding a build step
- Reporting broken Vercel deployment behavior

Before opening a pull request:

```powershell
npm run check
```

Please do not submit private classroom screenshots, copied paid course material, account data, or unverified answers.
